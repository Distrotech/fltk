class Fl_Wordwrap_Output

Class Hierarchy

    Fl_Output
   |
   +----Fl_Wordwrap_Output

Include Files

    #include <fltk/Fl_Wordwrap_Output.h>

Description

This widget is a subclass of Fl_Output that displays multiple lines of text, and wraps long lines of text into paragraphs. It also displays Tab ('\t') characters as whitespace to the next column rather than as ^I.
