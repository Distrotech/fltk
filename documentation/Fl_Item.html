#include <fltk/Fl_Item.h>

Description

This widget is designed to be put into Fl_Menu and Fl_Browser widgets. It draws it's label() without erasing the background, if FL_SELECTED is set in the flags it draws the label with the selected label color so it contrasts with the background.

Currently you can set type() to Fl_Item::TOGGLE to get a checkmark on the left and Fl_Item::RADIO to make it a radio button. This may change in future versions however and you may need a subclass for this.

In a multi-browser, the selected() indicates if the widget is currently selected.
