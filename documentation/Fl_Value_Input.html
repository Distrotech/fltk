Include Files

Description

Though this is a subclass of Fl_Valuator it is better thought of as a subclass of Fl_Float_Input (there is an internal Fl_Float_Input in there). The main difference is that value() returns a double instead of the text string.

If step() is greater than one an Fl_Int_Input is used instead.
