Description

The Fl_Menu_Window widget is a window type used for menus and tooltips. By default the window is drawn in the hardware overlay planes if they are available so that when the menu disappears whatever was behind it does not have to redraw. If this is not possible, on X it also sets the "save behind" flag to indicate that the server should save the image behind the menu.
