class Fl_Box

Class Hierarchy

Include Files

Description

This widget is the same as an plain Fl_Widget except if the box is set to FL_NO_BOX it is completely invisible and attemps to draw it do absolutely nothing. This can be used as the resizable() of a group or window to get more complex resize effects.

You can also set the box to provide decorations, but you can use the plain Fl_Widget for that, too.

Methods

Fl_Box::Fl_Box(int x, int y, int w, int h, const char * = 0)
Fl_Box::Fl_Box(Fl_Boxtype b, int x, int y, int w, int h, const char *)

 The first constructor sets box() to FL_NO_BOX, which means it is invisible. However such widgets are useful as placeholders or Fl_Group::resizable() values. To change the box to something visible, use box(n).

The second form of the constructor sets the box to the specified box type.

Fl_Box::~Fl_Box(void)

The destructor removes the box.
