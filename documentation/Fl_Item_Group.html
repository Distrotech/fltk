Description

This widget is designed to be put into Fl_Menu and Fl_Browser widgets. It indicates the title of a submenu, or a level of hierarchy in the browser. Any child widgets are the items in that submenu, or the items under this parent in the browser.

It draws it's label() without erasing the background, if FL_SELECTED is set in the flags it draws the label with the selected label color so it contrasts with the background.

In a browser, the value() indicates if the widget is open or not. In a multi-browser, the selected() indicates if the widget is currently selected.
