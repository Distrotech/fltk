Description

This widget describes a set of items that are to be put inside a Fl_Menu or Fl_Browser widget. It indicates the title of a submenu, or a level of hierarchy in the browser. Any child widgets are the items in that submenu, or the items under this parent in the browser.

If this widget is told to draw, it draws just like Fl_Item draws. See that for more details. The triangle indicating a submenu is not drawn by this, it is drawn by the parent menu.

Because this is an Fl_Menu_ subclass, you can also call popup() and add() and other methods to manipulate the items inside it.

In a browser, the value() indicates if the widget is open or not. In a multi-browser, the selected() indicates if the widget is currently selected.
