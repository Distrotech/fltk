Description

This input field displays '\n' characters as new lines rather than ^J, and accepts the Return, Tab, and up and down arrow keys, and wraps long lines to fit into paragraphs that fit in the window. This is useful for editing small blocks of text, but you may want Fl_Text_Editor instead.
