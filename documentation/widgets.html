Class Reference

Class Hierarchy

This appendix describes all of the classes and structures in FLTK.

The "class" Fl:: should actually be considered a namespace, it contains only static methods.

FLTK also provides a number of global functions with fl_xyz names. The majority of these are concerned with drawing or graphics state, and are described in the Drawing chapter. The others are a set of modal popup utility functions for asking questions described in the Utility Function chapter.
